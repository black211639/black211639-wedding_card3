# PROJECT_BRIEF.md

## 專案名稱

`wedding_card2`

## 專案目的

建立一個手機優先、正式、高級、溫柔、有婚禮感的 H5 婚禮電子喜帖。

## 使用者目標

這份電子喜帖會發給已確定會參加婚禮的賓客觀看。重點是讓賓客清楚知道婚禮時間、地點、交通資訊，並感受到正式邀請與溫柔氛圍。

## 技術架構

- 純 HTML
- 純 CSS
- 原生 JavaScript
- 靜態 JSON 資料：`wedding_info.json`
- 靜態素材：`assets/`

## 部署方式

GitHub Pages 根目錄部署。

## 不使用的東西

- React
