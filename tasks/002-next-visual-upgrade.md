# 002-next-visual-upgrade.md

## 目標

讓網站更像正式高級 H5 婚禮電子喜帖。

## 優先改善

1. Hero 首頁主視覺
2. 照片故事段落
3. 倒數區儀式感
4. 地圖與婚禮資訊整合
5. 結尾感謝頁

## 建議方向

- Hero 加強照片裁切、文字層級與遮罩。
- 照片故事段落減少模板感，讓照片更像主角。
- 倒數區改得更有婚禮儀式感，但保持可讀。
- 地圖與婚禮資訊可以更緊密，減少重複資訊。
- 結尾應溫柔、正式，適合正式喜帖收束。

## 驗收標準

- 手機看起來不像廉價模板。
- 照片不被錯誤裁切。
- 文字清楚。
- 功能不壞。
- 390px 寬度不爆版。
- `wedding_info.json` 與備援資料機制正常。
